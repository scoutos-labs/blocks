Blocks Skill Protocol — Draft 03: oracle capability, gate extension, blockHash split

PRD · slug blocks-protocol-draft03 · 2026-07-03 · scope FIXED (user-approved, 3 items) · status: plan approved

Definition

Draft 03 adds two small surfaces and corrects one preimage, keeping the co-signing symmetry and the honest tiers: (1) oracle.capability — a fuzzy contract names the oracle grade its rubric is calibrated against; oracles self-attest at record; truth is declared-tier. (2) Gate extension — a contains operator and a #ref length modifier, operators-not-functions, [GAT-7] ban list untouched. (3) blockHash preimage split — det blocks hash contract.json ‖ entry (prose is descriptive); fuzzy blocks unchanged (prose IS the contract); hashes become draft-scoped.
